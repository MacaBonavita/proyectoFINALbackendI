Nota: Se utilizará Node.js para ejecutar JavaScript JS
/**** Comandos ****/
-node app.js : Para ejecutar un script
-npm init -y : Nos crea un package de tipo JSON
-npm start : ejecuta el package
-node --watch app.js: Ejecución en espera de cambios (detecta cambios en el script por ejecutar)

Nota: Tener cuidado con el "auto-save" porque si se guarda un error mientras se ejecuta, se detendra la aplicación

-npm install : Instalar dependencia, librería, etc

/**** Comandos Teclas ****/
